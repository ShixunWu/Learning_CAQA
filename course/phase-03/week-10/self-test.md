## Q5：nop 与气泡

流水线中有 n 条指令，外加末尾 k-1 个 nop 排空流水线，总周期数公式是什么？如果指令数无限大，吞吐率趋近于什么？

---

## 答案

### A1
- 理想加速比 \( \frac{1000 \times 5}{5 + 1000 - 1} \approx 4.98x \)
- 有气泡：有效 CPI = 1 + 0.2 = 1.2，加速比 = 5/1.2 ≈ 4.17x，下降约 16%

### A2
- IF：从 I-Cache 取 lw 指令，PC+4
- ID：译码 lw，读寄存器 x2 的值
- EX：计算地址 = R[x2] + 0
- MEM：从数据存储器读地址处的值
- WB：将读出的值写入寄存器 x1

### A3
- 单周期时钟周期 = 5×200 = 1000ps
- 流水线时钟周期 = max(200) + 50 + 50 = 300ps
- 流水线时钟周期比单周期小（300ps vs 1000ps），但单条指令延迟变长（5×300=1500ps）

### A4
- 单周期：CPI=1, 时间=10×1000=10000ps
- 流水线：理想 CPI=1, 周期=300ps（取 max(200,150,250,300,100)+100=400ps），时间=(10+4)×400=5600ps
- 加速比=10000/5600≈1.79x

### A5
- 总周期数 = n + k - 1（k 为流水线级数）
- 吞吐率趋近于 1 条指令/周期（CPI→1）
