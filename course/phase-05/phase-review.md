# Phase 5 阶段复习：数据级并行 (Data-Level Parallelism)

> 覆盖教材：CAQA Chapter 4 + Appendix G，周 19-23

## 阶段测验（12 题，闭卷 45 分钟）

### Q1：向量执行时间
向量处理器 MVL=128，startup=20 周期，每元素 1 周期。处理 n=500 的向量加法（1 convoy）。求总执行时间。

### Q2：SIMD 元素计数
AVX-512（512-bit）寄存器可打包多少个 (a) float (b) double (c) int8？

### Q3：SSE 对齐检查
地址 `0x7fff_0028` 对 SSE 是否 16 字节对齐？对 AVX 是否 32 字节对齐？

### Q4：GPU Occupancy
SM 支持 2048 线程、64 warp。Block 大小 128，每线程 48 个寄存器（SM 共 65536 寄存器），无共享内存。求 Occupancy。

### Q5：Warp 分析
启动 `<<<512, 256>>>`。总共有多少个 warp？每个 SM 最多 64 warp，至少需要多少 SM 才能一次性容纳所有 warp？

### Q6：Bank Conflict
`__shared__ float sm[32][32]`，warp 访问 `sm[tid][tid]`。判断是否有 bank conflict，几-way？

### Q7：Roofline 分析
GPU：峰值 10 TFLOPS，带宽 800 GB/s。Kernel 算术强度 = 2 FLOP/Byte。能达到的最大 GFLOPS 是多少？瓶颈在哪？

### Q8：合并访问
warp 访问 `A[threadIdx.x * 2]`（stride=2）。事务粒度 32 字节。需要多少事务？带宽利用率？

### Q9：向量化判断
以下循环能否向量化？
```c
for (i=2; i<N; i++) a[i] = a[i-2] + b[i];
```

### Q10：Scatter/Gather
稀疏索引 15% 非零。n=10000，MVL=64，gather startup=15，element=1。gather 总时间？与标量（5 周期/元素）比较。

### Q11：Amdahl 定律（向量化）
程序 70% 向量化，向量加速比 10×。总体加速比？

### Q12：综合设计
需要 200W 内处理 N=128 的小矩阵乘法——选择向量/SIMD/GPU 并给出 2 个定量理由。

---

## 答案

### A1
ceil(500/128) = 4 轮。T = 4×20 + 500×1 = 80+500 = **580 周期**。

### A2
(a) 512/32=**16**, (b) 512/64=**8**, (c) 512/8=**64**。

### A3
0x28 = 40。40 mod 16 = 8 → 不对齐；40 mod 32 = 8 → 不对齐。

### A4
线程限制：2048/128=16 blocks；寄存器限制：65536/(128×48)=65536/6144=10.67→**10 blocks**。
Active blocks=10，threads=1280，Occupancy=1280/2048=**62.5%**。

### A5
Warp total = 512×256/32 = 512×8 = **4096 warps**。需要 SM 数 = ceil(4096/64)=**64 SM**。

### A6
bank(tid) = tid*32+tid = tid*33。33 mod 32 = 1，所以 bank = tid mod 32。连续线程访问不同 bank → **无冲突**。

### A7
上限 = min(10000, 800×2) = min(10000, 1600) = **1600 GFLOPS**，受**内存带宽**限制。

### A8
stride=2 → 线程 i 访问偏移 i×8 bytes。每连续 4 个线程占用 32B 事务（8+8+8+8=32 → 但数据交错！）。
实际上 32 线程跨越 32×2×4=256 字节 → 需要 **2 次 128B 事务**（若 128B 对齐）。带宽利用率 ≈ **50%**。

### A9
依赖距离 = 2。对向量化而言，只要距离 ≥ 向量宽度就安全。MVL=64 时 i-2 < i，存在 RAW 依赖但距离足够大 → **可向量化**（无迭代间真依赖在窗口内）。

### A10
轮次 = ceil(10000/64)=157。T_gather = 157×15 + 10000×1 = 2355+10000=**12355**。
标量：10000×5=50000。加速比 ≈ 4.0×。

### A11
S = 1/(0.3 + 0.7/10) = 1/(0.3+0.07) = 1/0.37 ≈ **2.70×**。

### A12
推荐**向量处理器**：1) N=128 接近 MVL=128-256 最佳范围；2) 功耗 < 30W 远低于 GPU 的 300W；3) GPU 对小矩阵 kernel launch 开销占比大。
